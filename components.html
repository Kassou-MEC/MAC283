// 01 · The Processor

CPU — The Brain and Heart

Definition · CPU (Central Processing Unit)

A small chip (microprocessor) that carries out instructions by performing arithmetic, logic, control, and I/O operations. Every operation done with the computer is processed by the CPU. It seats into a socket on the motherboard. Inside it: the ALU, Control Unit, Registers, Cache, and Clock.

// 02 · Where the CPU Lives

The CPU Socket

The CPU doesn't solder onto the board — it drops into a socket. The socket type decides which processors are compatible (you can't put an AMD chip in an Intel socket).

// 03 · Memory · Volatile

RAM — Random Access Memory

Definition · RAM

Volatile memory for temporary storage while programs run. It holds the data the CPU is actively processing. Power off → data gone. Key for multitasking and fast access. Types: DRAM, SRAM, DDR4, DDR5. It snaps vertically into DIMM slots near the CPU, often in pairs.

// 04 · Memory · Non-Volatile

Storage — SSD & HDD

Definition · Storage

Non-volatile memory: data remains after power is off. Stores files, software, and the OS permanently. Examples: HDD, SSD, USB, SD card. SSDs are faster and more durable than HDDs. Higher capacity than RAM. Mounted in a drive bay (SATA cable) or an M.2 slot.

VolatilityVolatileNon-volatile

// 05 · The Connector of Everything

The Motherboard

Definition · Motherboard

The main printed circuit board (PCB) of the computer. It holds the CPU, RAM, storage, and expansion slots, provides power and communication paths between components, and controls data flow with onboard chipsets. It is the base that connects everything.

// 06 · Expansion

Graphics Card

PCIe slots let you add capabilities after purchase — most famously the graphics card, which renders images to the display. A GPU is itself a non-von-Neumann machine: thousands of small processors working in parallel.

// 07 · The Bus

USB Ports & the System Bus

Definition · Bus

A pathway that transfers data between components — a group of wires where each wire carries 1 bit (0 or 1) at a time. Bus width = number of wires = bits moved at once (a 64-bit bus moves 64 bits per trip). USB = Universal Serial Bus, the external bus your peripherals plug into.

Bus laneCarries
Data busThe actual data between CPU, RAM, and devices
Address busMemory addresses — the "where"
Control busSignals like read/write and the clock — the "when & how"
