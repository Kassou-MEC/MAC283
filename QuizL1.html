    BAY 03

    Decode the Boot Message

    Data Representation15 pts

    A bootloader printed a 5-letter word as raw bytes. Each tile is one character in 8-bit binary. Read it the way the lecture taught: give each position its power of two, add up the on bits to get the ASCII decimal, then look up the letter. Type each letter to reveal the word.

    _ _ _ _ _
    Place values (right→left): 128 64 32 16 8 4 2 1. Uppercase letters run A=65 … Z=90.
    BAY 04

    Binary Forge

    Data Representation15 pts

    The ALU never adds in decimal. Do what the hardware does: convert both operands to binary, add them column by column, and report the result in binary plus how many carries the addition generated.

    25+ 13=
    A carry happens whenever a column sums to 2 or more and pushes a 1 into the next column left. Count every one.
