What is a Motherboard?

The main printed circuit board that everything else plugs into — the foundation that connects, powers, and coordinates every part of the machine.

Foundation

The main printed circuit board (PCB) of the computer.

Holds

Holds the CPU, RAM, storage, and expansion slots.

Connects

Provides power and communication paths between components.

Controls

Controls data flow through its onboard chipsets.

Component 02

What is a Bus?

A bus is a pathway used to transfer data between components — the network of electrical wires inside the system that interconnects the hardware.

Data Bus

Carries the actual information being moved.

Address Bus

Carries the location the data is going to.

Control Bus

Carries the timing & command signals.

Scope

Can be internal (inside the chassis) or external.

Examples

USB · PCIe · SATA · Front-Side Bus

Bus width · interactive

One wire, one bit

A bus is a group of cables tied together. Each cable carries 1 bit (binary 0 or 1) at a time — so a wider bus moves more bits per cycle. Bus width directly drives processor performance.

8cables / bits per cycle

An 8-bit bus has 8 cables, so it can transmit 8 bits of data at one time.

Reference

Roles & the bus system

Together, the motherboard and its buses connect and power components, coordinate communication, determine upgrade potential, and keep data flow fast and synchronized.

Wrap-up

What the board taught us

Motherboard

The PCB foundation — holds the CPU, RAM, and storage, and routes power & data between them.

Bus

Bundles of wires (each carrying 1 bit) that move data, addresses, and control signals. Wider = faster.

The cycle

Fetch → decode → execute: input lands in RAM, the CPU computes, and the result returns to the screen.

Architecture

Von Neumann shares one memory & bus (the bottleneck); Harvard splits them for parallel access.
