CH 01Introduction & Basic Computer Hardware LAB▶
Overview
  • The Main Components of a Computer

Basic Computer Hardware

CH 02Data Representation LAB▶
Data Representation
  • Binary, Octal, Decimal, Hexadecimal and Base conversion

  • Fixed Point and Fractions

CH 03Boolean Algebra, Combinational & Sequential Logic LAB▶
Boolean Algebra
  • Boolean Algebra
  • Truth Tables and K-Maps
Combinational Logic
  • Logic Gates
  • Building blocks: MUX, DEMUX, Encoder, Decoder, Adder…

Sequential Logic
  • Clocks, Flip-Flops
  • Hardwire design: Finite State Machine (Mealy, Moore)
  • Firmware design: Huffman model of sequential design (ROM + F/F)

  • The Bus, Clocks, I/O subsystems

  • Interrupts

Assemblers, Symbol Table, Scans/Passes
  • Writing Assembler code
Extending the Instruction Set
  • Hardwire vs Microprogramming controlled
  • Real-world examples
  • Extending MARIE's instruction set
CH 05Exploring Instruction Sets LAB▶
Exploring Instruction Sets
  • Format
  • Types
  • Addressing modes
Time Permitting
  • Privileged instructions
